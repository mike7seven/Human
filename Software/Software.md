# Software

Layer 2 of the Human System contains the "programs" that run on or with Human OS. These include tools, workflows, routines, automations, agents, and full systems.

## Software Categories

### Productivity & Task Management

Tools for managing tasks, reminders, and daily operations.

- Apple Reminders

### Knowledge Management & Note-Taking

Systems for capturing, organizing, and retrieving knowledge.

- Obsidian
- Apple Notes

### AI Agents & Assistants

AI-powered tools that augment cognitive capabilities and assist with various tasks.

- Claude
- Claude Code
- ChatGPT
- Codex
- Gemini

### Health & Fitness Tracking

Tools for monitoring physical well-being and fitness metrics.

- Apple Health/Fitness

### Financial Management

Systems for tracking finances, budgeting, and financial operations.

- Custom FinOps Software

### Communication & Collaboration

Tools for connecting and collaborating with others.

- Slack
- Discord

### Development & Coding Tools

Software development environments and programming tools.

- VSCode

### Automation & Workflows

Tools for creating automated routines and workflows.

- Apple Shortcuts

### Monitoring & Observability

Systems for tracking metrics, logs, and system health.

- TIG Stack (Telegraf, InfluxDB, Grafana)
- Grafana

### Creative Tools

Tools for creative work and content creation.

- *(To be defined)*

## Versioning

Each software component follows independent versioning per project. Recommended schemes:

- SemVer (recommended)
- Calendar versioning
- Build numbers
- Internal revision tracking
- Release channels (alpha/beta/stable)
